#  Product Data Cleaning & SEO Title Optimization
**A Data Analytics Stage 1 Project: Transforming Raw Marketing Data into Analysis-Ready Insights.**

##  Project Overview
In this project, I acted as a Data Analyst to clean and transform a "messy" raw product dataset using **Excel**. The primary goal was to resolve data quality issues and implement a logic-driven `short_title` feature to improve SEO and readability for marketing purposes.

##  Tools Used
* **Microsoft Excel:** Core tool for data cleaning and preparation.
* **Text Functions:** `TRIM`, `SUBSTITUTE`, `LEFT`, and Nested `IF` statements.
* **Data Validation:** Used for identifying inconsistencies and standardizing formats.

##  Key Objectives
1.  **Cleanse Data:** Identify and resolve missing values and duplicate records.
2.  **Standardize:** Normalize column headers (snake_case) and data formats.
3.  **Optimize:** Generate concise, SEO-friendly product titles (limit 30–50 characters).
4.  **Report:** Document technical steps and improvements in a structured format.

##  Data Cleaning Workflow
* **Handling Missing Values:** Evaluated incomplete records and applied strategies to ensure dataset integrity.
* **Duplicate Removal:** Identified and eliminated redundant entries to ensure each row represents a unique product.
* **Formatting:** Standardized column names (e.g., `sales_amount` instead of `SalesAmount`) and ensured uniform currency/date formats.

##  SEO Title Optimization (The "Short Title" Logic)
I developed a logic to strip redundant marketing phrases (e.g., "includes," "features," "set of") while retaining the **Brand**, **Product Type**, and **Key Attribute**.

| Original Title | Optimized Short Title |
| :--- | :--- |
| Tulip Flowers Blackout Curtain for Door, Window & Room | Tulip Blackout Curtain - 2 PCS |
| Marks & Spencer Girls' Pyjama Sets T86_2561C_Navy | Girls' Navy Pyjama Set - 9-10Y |

##  Results & Impact
* **Data Quality:** Reduced noise and errors by removing duplicates and fixing inconsistencies.
* **Business Value:** Prepared a dataset ready for immediate use in SEO marketing campaigns.
* **Scalability:** Implemented `IFERROR` and dynamic formulas to handle future data updates automatically.
